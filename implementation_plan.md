# แยกหน้า Dashboard และ My Packages

จากที่คุณแจ้งมา ผมเข้าใจแล้วครับว่าตอนที่แยก "แพ็กเกจของฉัน" ออกมาจาก "คำสั่งซื้อ" ผมเผลอเอาหน้า "แพ็กเกจของฉัน" ไปทับหน้า "แดชบอร์ดข้อสอบ" (`/exams`) เดิม ทำให้หน้าแดชบอร์ดเดิม (ที่มีส่วน Continue Learning และ เร็วๆ นี้) หายไป

ดังนั้นเราต้องมีทั้ง 2 หน้าแยกกันอย่างชัดเจน:

## 1. หน้า Dashboard (แดชบอร์ดข้อสอบ)
- **Route:** `/exams` (ใช้ URL เดิมที่เคยเป็น)
- **หน้าที่:** เป็นศูนย์กลางการเรียนรู้ มีส่วน "ทำข้อสอบต่อจากที่ค้างไว้" (Continue Learning), "ชุดข้อสอบล่าสุด", และส่วน placeholder เร็วๆ นี้ (Bookmarks, สถิติ)
- **การกู้คืน:** ผมจะกู้คืนโค้ดดั้งเดิมของหน้า `/exams` กลับมาเป็นแดชบอร์ดเหมือนก่อนหน้านี้

## 2. หน้า My Packages (แพ็กเกจของฉัน)
- **Route:** `/my-packages` (สร้าง Route ใหม่เพื่อไม่ให้ทับกับอะไร)
- **หน้าที่:** หน้ารวมแพ็กเกจทั้งหมดที่คุณเป็นเจ้าของ เพื่อเลือกเข้าเรียน (หน้าที่ผมเพิ่งสร้างให้ที่มี LearningCard + จำนวนข้อสอบ/สรุป)

## 3. การปรับ Navigation Links
- **Top Nav:** `แดชบอร์ด` -> ชี้ไปที่ `/exams`
- **Profile Dropdown (Desktop & Mobile):** 
  - `แดชบอร์ด` -> ชี้ไปที่ `/exams`
  - `แพ็กเกจของฉัน` -> ชี้ไปที่ `/my-packages` (แยกออกมาใหม่)
  - `คำสั่งซื้อ` -> ชี้ไปที่ `/orders`

## Open Questions
- คุณโอเคกับการใช้ URL `/my-packages` สำหรับหน้า **แพ็กเกจของฉัน** ไหมครับ? หรืออยากให้ใช้ URL อื่น (เช่น `/packages/mine`)
- โครงสร้างนี้ตรงกับที่คุณต้องการใช่ไหมครับ? หากโอเคแล้วกด Proceed หรือแจ้งยืนยันได้เลยครับ ผมจะดำเนินการแก้ไขให้ทันทีครับ
